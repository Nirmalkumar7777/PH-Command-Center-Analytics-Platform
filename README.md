📌 Overview

This project is an interactive Public Health Analytics Dashboard designed to analyze public health programs and population health outcomes. It helps in understanding disease trends, health disparities, resource allocation efficiency, and program effectiveness using data-driven insights.

The system integrates ETL processes, exploratory data analysis, segmentation, anomaly detection, forecasting, and KPI dashboards into a unified analytics platform.


---

🎯 Objectives

Analyze disease prevalence and health outcomes

Evaluate effectiveness of public health programs

Identify health disparities across regions and populations

Optimize resource allocation and policy decisions

Build predictive and anomaly detection models

Provide interactive visual dashboards for decision-making



---

🚀 Features

📊 Data Processing

Data cleaning and preprocessing (ETL pipeline)

Star schema-based data modeling

Missing value and outlier handling


📈 Analytics Modules

Exploratory Data Analysis (EDA)

Population segmentation (RFM-style analysis)

KPI monitoring dashboards

Fraud/anomaly detection system

Time series forecasting (future trend prediction)

Cohort & retention analysis


📉 Visualization

Interactive charts and dashboards

Health trend analysis by region and program

Disease distribution heatmaps

Resource allocation insights



---

🛠️ Tech Stack

Frontend: HTML, CSS, JavaScript

Backend: Python / Flask (or Replit Web Server)

Data Processing: Pandas, NumPy

Visualization: Plotly / Matplotlib / Power BI (optional)

Analytics: Scikit-learn, Statsmodels

Deployment: Replit
